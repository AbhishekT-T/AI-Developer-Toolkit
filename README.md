AI Developer Toolkit

An all-in-one AI-powered web application built with HTML, CSS, and JavaScript that helps developers and job seekers with coding, resumes, and prompt engineering.

The application supports both Google Gemini and Anthropic Claude APIs through a Bring Your Own API Key (BYOK) model, allowing users to securely use their own AI credentials.

Features
ATS Resume Generator
Resume vs Job Description Scanner with Match Score
Keyword Coverage & Resume Feedback
Multi-language Code Generator
AI Code Debugger with Explanations
SQL Query Generator
AI Prompt Generator
Prompt Analyzer & Optimizer
Supports Google Gemini and Anthropic Claude
Bring Your Own API Key (BYOK)

HTML5
CSS3
JavaScript
Google Gemini API
Anthropic Claude API
API Key

This project does not include API keys.

To use the application:

Obtain a Google Gemini API Key or an Anthropic Claude API Key.
Open AI Settings in the application.
Paste your API key.
Start using all AI-powered features.
